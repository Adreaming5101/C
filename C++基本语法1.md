```C++
%c和%s:
	/*
	都是在 C 和 C++ 中用于格式化字符串的占位符。
	%c 用于处理单个字符，在输入时，%c 从输入流中读取一个字符；在输出时，%c 将一个字符打印到输出流中。
	%s 用于处理字符串。在输入时，%s 从输入流中读取一串字符，直到遇到空格、制表符或换行符等空白字符为止，并将这些字符存储到一个以 null 结尾的字符数组中；在输出时，%s 打印该字符数组中的内容，直到遇到 null 终止符为止。
	区别和总结：
	%c 用于处理单个字符，而 %s 用于处理字符串（字符序列）。
	%c 只能处理一个字符，包括字母、数字、特殊字符等；%s 则可以处理一串字符，直到遇到空白字符或者字符串结尾。
	%c 在读取时不会跳过空白字符，而 %s 在读取时会跳过前导空白字符并且遇到空白字符会停止读取。
	%s 在处理字符串时需要使用字符数组来存储字符串，字符串以 null 结尾，而 %c 只是单个字符的处理，通常存储在一个字符变量中。
	*/
```

##### 自增与自检运算符

```c++
自增运算符与自减运算符：
    //自增自检运算符只能作用于整形变量以及字符型变量，不能作用于常量或运算表达式：如，5++，(a+b)++
    a++  <==> a = a + 1
    a--  <==> a = a - 1
    // a++、a-- 处理不了a + 2等情况，只能是a+1,a-1

    ++a 与 a++区别(自增运算符，a--  --a为自减运算符)：
        ++a 先自加，后参与运算;
        a++ 先参与运算，后自加;
    #include <stdio.h>
    int main() {
        int a = 8;
        int b = 0;
        b = 3 + a++;
        printf("%d\n", b); // b=11,a=9

        b = 3 + ++a;
        printf("%d", b);   // b=13,a=10
    }

    #include <stdio.h>
    int main() {
        int i = 5;
        int j;
        j = i++;
        printf("i=%d,j=%d\n", i, j);  // i=6,j=5
        printf("i=%d,j=%d\n", i++,j++);  // i=6,j=5，打印之后加
        printf("i=%d,j=%d", i, j);  // i=7,j=6
    }
```

##### 逻辑运算符

```c++
逻辑与：&&
逻辑或：||
逻辑非：!
一般形式：表达式 逻辑运算符 表达式
逻辑非的优先级高于算数运算符 ：
    int a = 5;
	int b = 7;
	int c;
	c = a + !b   // 5 + !7 = 5  7为真，非7为0


短路运算：
       逻辑与： 第一个表达式为假，不需要再运算第二个表达式，结果为假
       逻辑或： 第一个表达式为真，不需要再运算第二个表达式，结果为真
```

##### 逗号运算符

```c++
逗号运算符(又称顺序求值运算符)：可以把表达式连接起来，整个逗号表达式的值为最后一个表达式的值，逗号运算符的优先级是所有运算符中最低的。如，
    #include <stdio.h>
    int main() {
        int a = 3;
        a = 3*5,a*4;
        printf("%d", (3*5,a*4));  // 表达式的值为60
        return 0;
    }
```

##### sizeof()运算符

```c++
用来获取数据类型大小的运算符。它返回指定类型或变量的字节数。
    1.用于数据类型
   	  sizeof使用形式：sizeof(int)
      数据类型必须用括号括住
    2.用于变量
      使用形式：sizeof(a)
sizeof操作符不能用于函数类型，不完全类型（未知存储大小的数据类型）或位字段。void

sizeof()直接参与运算：
    int i = 10;
	i * sizeof(int)
        
使用场景：
内存分配： 在动态内存分配时，可以使用 sizeof 来确定分配内存的大小，特别是在使用 malloc 或 calloc 分配内存时，以确保分配足够的空间。
数组操作： 对数组进行操作时，sizeof 可以帮助计算数组元素的个数，这对于循环或者动态数组的操作非常有用。
跨平台编程： 在编写可移植的代码时，使用 sizeof 可以确保数据类型在不同平台上具有相同的大小。
   	int *ptr;
	ptr = (int *)malloc(5 * sizeof(int)); // 分配足够空间来存储5个整数

    
```

##### 运算符的优先级与结合性

```c++
由低到高分别是：
    逗号运算符 < 赋值运算符 < 条件运算符 < 逻辑运算符 < 位运算符 < 关系运算符 < 算数运算符 < 特殊运算符

大多数运算符从左往右执行，例如：
    a++, a-b+c, a%b, a>=b

    
右结合性(从右往左执行)：
    a = b = c = 5    // 将5赋值给c再赋值给b再赋值给a
	前增量：++a
    前减量：--a
    逻辑非：!
    正负号
    取地址：&
    sizeof
    
补充：
    // 找优先级最低的，为表达式名称
    a = b = c =5    //赋值表达式 
    b + c,c + d     //逗号表达式
    a > b || c      //逻辑表达式
    a > b + c       // 关系表达式
    a + b           //算数表达式
    
    
    int a,b,c,d;
	int e = (a=1,b=a++,c=a*b,d=c++ +b);
	int f = (a=1,b=++a,c=a*b,d=c+ ++b);
```
